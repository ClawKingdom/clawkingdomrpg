6. COMBAT SYSTEM

Turn-based, oracle-seeded combat against procedural bosses.

  • Hero attacks: Damage = ATK + Oracle RNG(0-50%) + CRIT_BONUS
  • Boss retaliates: Damage = Boss_ATK + RNG(0-30%)
  • Repeat: Until one reaches 0 HP

Victory = Rewards

  • WIN: Full XP + $VOID from treasury + 1-3 loot drops
  • LOSS: 25% XP (consolation) + 0 $VOID

Stat Growth

  • Per level: STR +2, DEX +2, INT +2, VIT +1.5
  • No soft caps, linear scaling
